# todo-list

This is a todo-list app implement with React Native

## Getting Started

First just

```
git clone https://github.com/log1NVolcom/todoList.git
```

Then

```
npm install
```

Finally

```
exp start
```

Now its ready to use

### Prerequisites

You need to install Node.js

### Sizes

24px for header of google
